## 📐 技术实现细节

### **单轴跟踪算法：**
```javascript
1. 计算太阳方位角
2. 限制旋转范围（±60°）
3. 应用每帧最大角度限制（0.57°）
4. Lerp插值平滑过渡（0.8%）
5. 夜间模式切换
```

### **阴影优化：**
```javascript
directionalLight {
  intensity: 3
  castShadow: true
  shadow-mapSize: 2048×2048
  shadow-bias: -0.0001
}

mesh {
  castShadow: true  // 光伏板
  receiveShadow: true  // 地面
}
```

### **地形生成：**
```javascript
PlaneGeometry(100, 80, 50, 40)
- 坡度效果：-x * 0.05
- 起伏效果：sin(x)*cos(y) * 1.2
- 双面渲染
- 绿色草地材质
```
