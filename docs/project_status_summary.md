# PvSimulator 项目状态总结

📅 **更新日期**：2025-11-05

## 🎯 项目概述

PvSimulator 是一个基于企业真实地形数据的光伏发电模拟系统，实现了论文级别的 **Terrain-Aware Backtracking** 算法，用于准确预测光伏系统在复杂地形下的发电性能。

### 核心特性
- ✅ **真实地形数据**：使用企业提供的403排跟踪行、3779个桩位的真实数据
- ✅ **论文算法实现**：完全匹配后端和前端的地形感知回溯算法
- ✅ **3D可视化**：基于Three.js的实时追踪轨迹和遮挡状态展示
- ✅ **高性能API**：支持分页、过滤、抽样和聚合的数据接口
- ✅ **健壮的后端**：缓存锁、异常重试、降级回退机制

### 阶段 10：3D场景升级 ⭐ **今日完成**
**前端算法完全匹配论文和后端实现**：

#### 核心改进
1. **GCR计算**：`moduleWidth / rowPitch`，限制在0.05-0.9
2. **遮挡裕度**：`margin = solar_elevation - blocking_angle`
3. **坡度补偿**：基于高度差估算坡度修正
4. **沿轴距离衰减**：20%衰减因子（论文关键参数）
5. **邻居过滤**：
   - 横向距离：0.5-20米
   - 沿轴距离：≤250米
6. **太阳侧判定**：只考虑太阳方向的邻居

## 📊 技术栈

### 后端
- **框架**：FastAPI
- **数据库**：SQLite（开发）/ PostgreSQL（生产）
- **核心库**：pvlib, pandas, numpy
- **测试**：Pytest

### 前端
- **框架**：React
- **3D渲染**：Three.js + @react-three/fiber
- **UI库**：Ant Design
- **图表**：Recharts

### 算法
- **论文**：Terrain-Aware Backtracking via Forward Ray Tracing
- **实现语言**：Python（后端）+ JavaScript（前端）
- **核心参数**：
  - GCR（地面覆盖率）
  - 遮挡裕度（Shading Margin）
  - 沿轴衰减因子（20%）
  - 坡度补偿
