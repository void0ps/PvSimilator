## 📖 论文2核心算法详解

### 1. Forward Ray Tracing方法

#### 论文描述（第1页）：
```
"Looking at each time step individually one can set up the 
backtracking problem as a traditional forward ray tracer with 
one screen object necessary through which to trace rays. In the 
simplest incarnation of a backwards ray tracing program such as 
those used in the movie and video game industries, rays are 
traced from a "camera", through the pixels on a screen and into 
the scene and then finally towards a light source."

从太阳位置发射射线 → 检测是否击中bay → 判断是否被遮挡
```

#### 论文算法流程：
```
1. 为每个bay创建一个平面（plane with normal vector）
2. 从太阳方向发射射线到每个bay
3. 检测射线是否通过bay后击中其他bay
4. 如果击中，则两个bay都需要回溯直到无遮挡
5. 回溯步长：1度
```

#### 当前实现：
```python
# terrain_backtracking.py
```

- ❌ **完全未实现射线追踪**
- ✅ 使用几何近似代替（合理简化）
- ⚠️ 对复杂地形可能不够精确

#### 加权平均实现
```python
def weighted_average_poa(
    bay_poa: Dict[int, pd.Series],
    bays: List[Bay]
) -> pd.Series:
    """按模块数量加权平均POA"""
    total_modules = sum(bay.module_count for bay in bays)
    weighted_poa = pd.Series(0.0, index=bay_poa[bays[0].bay_id].index)
    
    for bay in bays:
        weight = bay.module_count / total_modules
        weighted_poa += bay_poa[bay.bay_id] * weight
    
    return weighted_poa
```

### 优先级2：改进遮挡因子模型

#### 论文的二值模型

```python
def shading_factor_binary(margin: float, diffuse_fraction: float = 0.38):
    """论文的简化二值模型"""
    if margin >= 0:
```

#### 我们当前的渐进模型（可能更合理）
```python
def shading_factor_progressive(margin: float, diffuse_fraction: float = 0.38):
    """渐进过渡模型"""
    if margin >= 0:
        return 1.0
    neg_margin = abs(margin)
    if neg_margin <= 2.0:
        return 1.0 - 0.1 * (neg_margin / 2.0)
    elif neg_margin <= 10.0:
        ratio = (neg_margin - 2.0) / 8.0
        return 0.9 * (1.0 - ratio) + diffuse_fraction * ratio
    else:
        return diffuse_fraction
```

**建议**：提供两种模式供用户选择
