### 2. 遮挡数据接口优化 (`simulations.py`)

#### 2.1 添加分页功能
**新增参数**：
- `limit`: 返回记录数限制（1-10000）
- `offset`: 偏移量

**返回信息**：
```json
{
  "pagination": {
    "total": 1440,
    "offset": 0,
    "limit": 100,
    "returned": 100,
    "sample_rate": null
  }
}
```

#### 2.2 时间段过滤
**新增参数**：
- `start_time`: 开始时间（ISO格式）
- `end_time`: 结束时间（ISO格式）

**示例**：
```
GET /api/v1/simulations/1/shading?start_time=2025-11-01T00:00:00&end_time=2025-11-01T23:59:59
```

#### 2.3 数据抽样
**新增参数**：
- `sample_rate`: 抽样率（1-100）

**功能**：
- `sample_rate=5` 表示每5条取1条
- 大幅减少数据传输量
- 适合绘制趋势图

#### 2.4 新增聚合接口
**路径**：`GET /api/v1/simulations/{id}/shading/aggregated`

**参数**：
- `interval`: 时间间隔（如：`1H`=每小时，`1D`=每天，`15T`=每15分钟）
- `metric`: 聚合指标（`mean`/`min`/`max`/`median`）

**返回示例**：
```json
{
  "simulation_id": 1,
  "include_shading": true,
  "series": [...],
  "count": 72,
  "aggregation": {
    "interval": "1H",
    "metric": "mean",
    "original_count": 1440,
    "reduction_ratio": "95.0%"
  }
}
```

**优势**：
- 将1440条数据聚合为72条（小时级）
- 数据量减少95%
- 保留统计特征
- 适合长周期模拟
