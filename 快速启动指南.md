# ⚡ PV Simulator 快速启动指南

## 🚀 最简单的启动方式（推荐）

### 双击运行这个文件：
```
启动系统-简易版.bat
```

这个脚本会自动：
1. ✅ 在独立窗口启动后端服务
2. ✅ 在独立窗口启动前端服务
3. ✅ 显示访问地址

---

## 📋 手动启动（如果自动启动失败）

### 步骤1：启动后端

**打开第一个CMD窗口**，执行：
```cmd
cd backend
venv\Scripts\activate
python main.py
```

看到这个提示表示成功：
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

---

### 步骤2：启动前端

**打开第二个CMD窗口**，执行：
```cmd
cd frontend\analysis
npm run dev
```

看到这个提示表示成功：
```
➜  Local:   http://localhost:3000/
```

---

## ❌ 常见问题

### 问题1：PowerShell脚本无法运行

**错误信息**：
```
无法加载文件...因为在此系统上禁止运行脚本
```

**解决方案（3选1）**：

#### 方案A：使用BAT文件（推荐）
```cmd
启动系统-简易版.bat
```

#### 方案B：临时绕过限制
```powershell
powershell -ExecutionPolicy Bypass -File .\启动后端.ps1
```

#### 方案C：修改执行策略（需要管理员权限）
```powershell
# 以管理员身份运行PowerShell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

---

### 问题2：后端启动失败

**错误**：`ModuleNotFoundError: No module named 'fastapi'`

**解决**：
```cmd
cd backend
pip install -r requirements.txt
```

---

### 问题3：前端启动失败

**错误**：依赖缺失

**解决**：
```cmd
cd frontend\analysis
npm install
```

---

### 问题4：端口被占用

**后端端口(8000)被占用**：
```cmd
netstat -ano | findstr :8000
taskkill /PID <进程ID> /F
```

**前端端口(3000)被占用**：
```cmd
netstat -ano | findstr :3000
taskkill /PID <进程ID> /F
```

---

## 🛑 停止服务

### 如果使用了自动启动脚本：
- 关闭对应的服务窗口即可

### 如果手动启动：
- 在服务运行的窗口按 `Ctrl + C`
