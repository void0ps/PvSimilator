## 🛠️ 常见问题速查

### ❌ 数据库问题

**问题**：`unable to open database file`

**解决**：
```cmd
cd D:\code\PvSimilator\backend
python -c "import sqlite3; sqlite3.connect('pv_simulator.db').close(); print('OK')"
```

---

### ❌ 模块未找到

**问题**：`ModuleNotFoundError: No module named 'xxx'`

**解决**：
```cmd
cd backend
pip install -r requirements.txt
```

---

### ❌ 端口被占用

**问题**：`Address already in use: 8000`

**解决**：
```cmd
# 查找进程
netstat -ano | findstr :8000

# 终止进程（替换<PID>）
taskkill /PID <PID> /F
```

---

### ❌ PowerShell脚本无法运行

**问题**：`无法加载文件...禁止运行脚本`

**解决（3选1）**：

**方案1：使用BAT文件**
```cmd
# 不需要修改策略
启动后端.bat
```

**方案2：临时绕过**
```powershell
powershell -ExecutionPolicy Bypass -File .\启动后端.ps1
```

**方案3：修改策略**
```powershell
# 管理员权限运行
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

---

### ❌ Node.js/npm未安装

**问题**：`'npm' 不是内部或外部命令`

**解决**：

4. 重启命令行
5. 验证：`node --version`

**临时方案**：只启动后端，暂时不用前端

## 📦 完整环境配置

### Python环境（后端）

```cmd
# 1. 检查Python版本
python --version  # 应该是3.11+

# 2. 进入后端目录
cd D:\code\PvSimilator\backend

# 3. 创建虚拟环境（如果没有）
python -m venv venv

# 4. 激活虚拟环境
venv\Scripts\activate

# 5. 安装依赖
pip install -r requirements.txt

# 6. 初始化数据库
python -c "import sqlite3; sqlite3.connect('pv_simulator.db').close()"

# 7. 启动服务
python main.py
```

### Node.js环境（前端）

```cmd
# 1. 安装Node.js
# 从 https://nodejs.org/ 下载并安装

# 2. 检查安装
node --version
npm --version

# 3. 进入前端目录
cd D:\code\PvSimilator\frontend\analysis

# 4. 安装依赖
npm install

# 5. 启动开发服务器
npm run dev
```
