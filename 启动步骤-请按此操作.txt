==============================================================
  PV Simulator 启动步骤 - 请按照以下步骤操作
==============================================================

✅ 后端已自动启动！

现在请按照以下步骤启动前端：

--------------------------------------------------------------
步骤1：打开一个新的CMD窗口（重要！）
--------------------------------------------------------------

方法1：按 Win + R，输入 cmd，回车
方法2：在开始菜单搜索"命令提示符"

注意：必须是新窗口，以确保环境变量生效

--------------------------------------------------------------
步骤2：在新CMD窗口中执行以下命令
--------------------------------------------------------------

cd D:\code\PvSimilator\frontend\analysis
npm install
npm run dev

--------------------------------------------------------------
步骤3：等待前端启动
--------------------------------------------------------------

看到以下提示表示成功：
  ➜  Local:   http://localhost:3000/

--------------------------------------------------------------
步骤4：访问系统
--------------------------------------------------------------

前端：http://localhost:3000
后端：http://localhost:8000
API文档：http://localhost:8000/docs

==============================================================

如果npm命令仍然不可用，请：
1. 完全关闭所有CMD/PowerShell窗口
2. 重新打开一个新的CMD窗口
3. 输入：npm --version（验证npm可用）
4. 然后再执行上面的步骤2

==============================================================

【快捷命令 - 复制粘贴使用】

cd D:\code\PvSimilator\frontend\analysis && npm install && npm run dev
