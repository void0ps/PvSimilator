# 🚀 PV Simulator 完整启动指南

## ⚡ 快速启动（2个窗口）

### 窗口1：启动后端

1. **打开CMD窗口**（按 Win+R，输入 cmd，回车）
2. **执行以下命令**：

```cmd
cd D:\code\PvSimilator\backend
venv\Scripts\activate
python main.py
```

3. **看到以下提示表示成功**：
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

---

### 窗口2：启动前端

1. **再打开一个新的CMD窗口**（重要：必须是新窗口！）
2. **验证npm可用**：
```cmd
npm --version
```
   如果看到版本号（如 9.x.x），说明npm可用，继续下一步
   如果提示"不是内部命令"，请完全关闭所有终端，重新打开

3. **执行以下命令**：
```cmd
cd D:\code\PvSimilator\frontend\analysis
npm install
npm run dev
```

4. **看到以下提示表示成功**：
```
➜  Local:   http://localhost:3000/
```

---

## ❗ 重要提示

### 关于npm环境变量

**问题**：npm命令不可用

**原因**：环境变量需要在新终端中才生效

**解决方法**：
1. 完全关闭所有CMD/PowerShell窗口
2. 重新打开一个新的CMD窗口
3. 测试：`npm --version`
4. 如果仍然不行，重启电脑（确保环境变量生效）

## 🔍 故障排查

### 问题1：后端启动失败

**错误**：`unable to open database file`

**解决**：数据库文件已创建，应该不会有这个问题。如果出现，运行：
```cmd
cd D:\code\PvSimilator\backend
python -c "import sqlite3; sqlite3.connect('pv_simulator.db').close()"
```

---

### 问题2：npm命令不可用

**错误**：`'npm' 不是内部或外部命令`

**解决步骤**：

1. **验证Node.js安装路径**
   - 默认路径：`C:\Program Files\nodejs\`
   - 检查此目录是否存在

2. **检查环境变量**
   - 按 Win 键，搜索"环境变量"
   - 点击"编辑系统环境变量"
   - 点击"环境变量"按钮
   - 在"系统变量"中找到"Path"
   - 确认包含：`C:\Program Files\nodejs\`

3. **重启终端**
   - 完全关闭所有CMD/PowerShell
   - 重新打开一个新窗口
   - 测试：`npm --version`

4. **如果还是不行**
   - 重启电脑
   - 重新安装Node.js（确保勾选"Add to PATH"选项）

---

### 问题3：前端依赖安装失败

**错误**：`npm install` 报错

**解决**：
```cmd
# 清理缓存
npm cache clean --force

# 删除旧的node_modules
cd D:\code\PvSimilator\frontend\analysis
rmdir /s /q node_modules

# 重新安装
npm install
```

---

### 问题4：端口被占用

**后端端口8000被占用**：
```cmd
netstat -ano | findstr :8000
taskkill /PID <进程ID> /F
```

**前端端口3000被占用**：
```cmd
netstat -ano | findstr :3000
taskkill /PID <进程ID> /F
```
